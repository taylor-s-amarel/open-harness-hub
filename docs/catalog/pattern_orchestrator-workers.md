# Orchestrator-Workers

*pattern* · `pattern/orchestrator-workers` · v0.1.0 · stable

A central orchestrator decomposes the task and delegates parallel
subtasks to worker agents (sometimes with different personas / tools
per worker). Workers return structured results; orchestrator
aggregates. Differs from ReAct because workers operate in parallel
with isolated contexts.
