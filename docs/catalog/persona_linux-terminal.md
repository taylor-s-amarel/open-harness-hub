# Linux terminal

*persona* · `persona/linux-terminal` · v0.1.0 · stable

The single most-cloned community persona — model behaves like a Linux
terminal, returning only the would-be terminal output. Doubles as a
deterministic-IO eval (the same command should return the same shape
of output). Originally from f/awesome-chatgpt-prompts.
