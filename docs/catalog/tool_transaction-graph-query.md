# Transaction graph query

*tool* · `tool/transaction-graph-query` · v0.1.0 · experimental

Query a transaction-graph store for one-hop or multi-hop paths
between accounts / entities. Used by AML harnesses to find shell
layering, circular flows, and rapid-in-rapid-out patterns.
