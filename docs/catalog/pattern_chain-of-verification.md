# Chain-of-Verification (CoVe)

*pattern* · `pattern/chain-of-verification` · v0.1.0 · stable

Generate a draft response, decompose it into atomic verifiable
claims, execute independent verification queries per claim, and
emit a final response that drops or revises unverifiable claims.
Reduces hallucination by independently re-checking each claim
against retrieval (rather than self-reflecting against the same
reasoning that produced the hallucination).
