# ESG social (S) sub-rubric v1

*rubric* · `rubric/esg-social-v1` · v0.1.0 · experimental

Six-dimension rubric for grading ONLY the social dimension of a
supplier disclosure. Aligned to ILO 11 forced-labor indicators,
CSRD ESRS S1 (own workforce) + S2 (value-chain workers), and
ILO core conventions (29, 87, 98, 100, 105, 111, 138, 182).

Use this when you want focused S-grading; use
`rubric/esg-supplier-compliance-v1` for combined E+S+G grading.
