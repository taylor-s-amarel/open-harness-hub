# Cost meter

*processor* · `processor/cost-meter` · v0.1.0 · stable

Emit per-call USD cost accounting given (adapter_ref, input_tokens,
output_tokens, cached_tokens). Resolves the adapter's pricing card,
multiplies, and writes a metering row to the configured sink.
