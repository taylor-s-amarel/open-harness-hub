# BGE embeddings (local)

*adapter* · `adapter/bge-embeddings-local` · v0.1.0 · stable

Local BGE (BAAI General Embeddings) via sentence-transformers. The
small variant is 384-dim and runs on CPU; the large variant is
1024-dim and benefits from GPU. Strong English + multilingual
baselines; no API key needed.
