# Sanctions list check

*tool* · `tool/sanctions-check` · v0.1.0 · stable

Check a normalized entity name against one or more sanctions lists
(OFAC SDN, UN Consolidated, EU Consolidated, HMT, or institutional
PEP). Returns matches above the configured fuzz threshold.
