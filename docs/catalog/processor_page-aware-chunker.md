# Page-aware chunker

*processor* · `processor/page-aware-chunker` · v0.1.0 · experimental

Split page-extracted text into ~800-token chunks with 120-token overlap; preserve the (page, span) coordinates on every chunk so downstream answers can cite back to the page + span.
