# Self-consistency sampler

*processor* · `processor/self-consistency-sampler` · v0.1.0 · beta

Run N parallel samples of a chain-of-thought reasoning prompt, then
majority-vote on the final answer (Wang et al. 2023). Robust to
arithmetic and reasoning errors that a single sample would miss.
