# GO/NO-GO checklist gate

*pattern* · `pattern/go-nogo-checklist-gate` · v0.1.0 · beta

Wrap a candidate action with an explicit checklist whose unmet items hard-gate the action. Models the human-performance discipline that real-world surgical teams, flight crews, mine examiners, and refinery start-up reviewers use: do not proceed unless every required item is verified, AND any item meeting a NO-GO trigger blocks the action regardless of model confidence. Reusable across any high-consequence pipeline (medical, aerospace, energy, finance, infra).
