# RAG-Fusion

*pattern* · `pattern/rag-fusion` · v0.1.0 · stable

Generate N rewrites of the user query, run retrieval for each, and
fuse the ranked lists via Reciprocal Rank Fusion. Outperforms single-
query retrieval when the user's wording differs from the corpus's.
