# Google Maps Geocoding (address → lat/lng)

*tool* · `tool/google-geocode` · v0.1.0 · stable

Forward and reverse geocoding via Google Maps Geocoding API.
Returns lat/lng + place_id + formatted_address + address_components
(street_number / route / locality / admin_area_level_1 / country).

Requires GOOGLE_MAPS_API_KEY env var. Free tier: ~$0 up to $200/mo.
