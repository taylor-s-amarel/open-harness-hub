# Inbox Zero Coach (triage + reply-draft + delegation)

*persona* · `persona/inbox-zero-coach` · v0.1.0 · experimental

Email triage assistant that classifies each thread (do-now /
delegate / defer / drop) and drafts reply candidates using the
user's prior tone with each correspondent.
