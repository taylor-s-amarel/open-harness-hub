# Calendar slot finder (preference-aware)

*processor* · `processor/calendar-slot-finder` · v0.1.0 · experimental

Given the user's calendar + their preferences (no-meeting hours,
focus blocks, preferred meeting hours), find N candidate slots
for a new meeting. Honors duration, time-zone, attendee
availability, and the user's hard rules.

Refuses to suggest slots in no-meeting windows unless the
caller has explicitly authorized override.
