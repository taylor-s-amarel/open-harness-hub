# Source document → persistent knowledge layer

*pattern* · `pattern/source-document-to-persistent-knowledge-layer` · v0.1.0 · beta

Architectural pattern: instead of treating each chat turn as ephemeral, encode the source document as a long-lived knowledge layer (wiki pages + concept graph + study guide + bookmarks + notes) stored in a local DB. The chat becomes a thin UI on top of this layer rather than the layer itself. Generalizes from CiteMind (PDF) to compliance audit notes, regulatory frameworks, supply-chain disclosures.
