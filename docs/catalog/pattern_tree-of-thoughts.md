# Tree of Thoughts (ToT)

*pattern* · `pattern/tree-of-thoughts` · v0.1.0 · beta

At each reasoning step, propose N candidate "thoughts", value/vote
each, and expand the best ones via BFS or DFS. Backtrack on dead-ends.
Generalizes chain-of-thought from a single chain to a tree, allowing
exploration of multiple reasoning paths.
