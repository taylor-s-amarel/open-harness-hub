# RAFT (Retrieval-Augmented Fine-Tuning)

*pattern* · `pattern/raft-retrieval-aug-finetune` · v0.1.0 · beta

Fine-tune an LLM on (question, retrieved_docs_including_distractors,
reasoning_chain, gold_answer) tuples so the model learns to USE the
retrieved docs at inference time AND ignore distractors. Bridges
the gap between pure RAG (no model training) and pure fine-tuning
(no retrieval). Substantially better domain Q&A vs either alone.
