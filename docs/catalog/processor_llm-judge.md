# LLM-as-judge

*processor* · `processor/llm-judge` · v0.1.0 · beta

Generic LLM-as-judge wrapper. Given (candidate response, rubric,
context), returns a per-dimension score with rationale and a
weighted-sum overall score. Independent of the model under review —
the judge sits outside that model's reasoning trace.
