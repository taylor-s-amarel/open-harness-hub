# OpenAI embeddings (text-embedding-3-small/large)

*adapter* · `adapter/openai-embeddings` · v0.1.0 · stable

OpenAI-compatible embeddings adapter. Defaults to
text-embedding-3-small (1536-dim, $0.02 / 1M tokens). Set
OPENAI_API_KEY at runtime. Works against the OpenAI API directly,
Azure OpenAI, or any OpenAI-compatible endpoint (Together, Groq,
Anyscale, vLLM, etc.) by overriding the endpoint.
