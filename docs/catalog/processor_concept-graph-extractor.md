# Concept graph extractor

*processor* · `processor/concept-graph-extractor` · v0.1.0 · experimental

Extract typed concept-graph nodes + edges from chunked text. Node types: concept, term, person, dataset, method, equation. Edge types: depends_on, cited_by, sub_concept, contradicts, defined_in. Each node + edge carries a (page, span) source anchor.
