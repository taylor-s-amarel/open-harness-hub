# Cost ceiling gate

*processor* · `processor/cost-ceiling-gate` · v0.1.0 · beta

Reject a pipeline run if the predicted USD cost exceeds the budget
configured for the calling pipeline / user. Uses token-count
estimates + adapter pricing to predict cost.
