# PDF extract with OCR fallback (CiteMind shape)

*processor* · `processor/pdf-extract-with-ocr-fallback` · v0.1.0 · experimental

Extract embedded PDF text page by page; fall back to OCR (Tesseract / PaddleOCR / similar) for pages whose embedded-text yield is below threshold (scanned pages, image-heavy figures).
