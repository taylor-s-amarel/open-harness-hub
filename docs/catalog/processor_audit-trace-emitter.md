# Audit trace emitter

*processor* · `processor/audit-trace-emitter` · v0.1.0 · stable

Emit a per-step audit trace including hashed input/output, model
version, adapter, applied layers, latency, and cost. Writes to a
configured sink (local JSONL, Loki, OpenTelemetry, SIEM).
