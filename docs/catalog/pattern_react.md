# ReAct (Reason + Act)

*pattern* · `pattern/react` · v0.1.0 · stable

Interleave reasoning steps with tool calls. At each step the model
emits one of: Thought (free-form reasoning), Action (tool call),
Observation (tool result fed back). Loop until the model emits a
Finish action. The canonical agentic loop.
