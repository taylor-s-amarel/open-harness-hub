# PDF to text

*processor* · `processor/pdf-to-text` · v0.1.0 · beta

Convert a PDF (extractable layer + optional OCR fallback) into plain
text with page breaks preserved. Returns text plus per-page byte
offsets so downstream chunkers can attribute chunks back to pages.
