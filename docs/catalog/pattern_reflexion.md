# Reflexion

*pattern* · `pattern/reflexion` · v0.1.0 · stable

Add a verbal self-reflection step after each failed attempt. The
reflection is appended to memory and shown to the model on the next
attempt. Bounded by `max_iters`. Captures the "I should have tried
X next time" insight that improves performance on subsequent
attempts without weight updates.
