# Recursive character chunker

*processor* · `processor/recursive-character-chunker` · v0.1.0 · stable

Split text into chunks using a recursive character splitter
(LangChain-style) that prefers paragraph → sentence → word
boundaries. Returns chunks with overlap + per-chunk byte offsets so
citations can point back to the source.
