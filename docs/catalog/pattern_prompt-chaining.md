# Prompt Chaining

*pattern* · `pattern/prompt-chaining` · v0.1.0 · stable

Break a task into a SEQUENCE of LLM calls where each call's output is
the next call's input. The simplest agent pattern — no tools, no
loops, no routing — just a linear chain. Useful when a task is too
complex for one prompt but doesn't need agency.
