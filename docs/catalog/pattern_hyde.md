# HyDE (Hypothetical Document Embeddings)

*pattern* · `pattern/hyde` · v0.1.0 · stable

Generate a hypothetical answer to the query (allowing hallucination),
then embed THAT answer instead of (or alongside) the query, and retrieve
against the embedding. Reduces query-document vocabulary gap because
the model's hallucinated answer is in the same "register" as real
source documents.
