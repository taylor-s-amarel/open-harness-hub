# Concept graph from text

*pattern* · `pattern/concept-graph-from-text` · v0.1.0 · beta

Promote relationships between entities mentioned in source text to first-class graph nodes + edges. Each node + edge is source-anchored (page, span). Generalizes from CiteMind's Book Wiki concept graph to ESG materiality maps, supply-chain knowledge graphs, regulatory-framework cross-reference graphs, drug-interaction graphs.
