# Citation coverage verifier

*processor* · `processor/citation-coverage` · v0.1.0 · beta

Verify that every factual sentence in a response carries at least
one citation marker (e.g. `[1]`, `[smith-2026]`). Returns coverage
ratio and a list of uncited sentences for re-prompt.
