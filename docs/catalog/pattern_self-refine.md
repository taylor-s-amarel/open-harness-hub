# Self-Refine

*pattern* · `pattern/self-refine` · v0.1.0 · stable

Single critique-and-revise pair. The model drafts an answer, then
critiques its own draft against a rubric, then revises. One iteration
by default; can be looped. Simpler than Reflexion (no memory store
across attempts) and Constitutional-AI (no fixed critique principles).
