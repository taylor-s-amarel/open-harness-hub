# ICD-10 lookup

*tool* · `tool/lookup-icd10` · v0.1.0 · stable

Lookup an ICD-10 diagnosis code by code or label substring. Returns
code + label + category. Backend-agnostic; intended to bind to a
local copy of the WHO ICD-10 release or to an institutional
terminology server.
