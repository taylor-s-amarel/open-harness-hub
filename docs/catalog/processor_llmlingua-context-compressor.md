# LLMLingua context compressor

*processor* · `processor/llmlingua-context-compressor` · v0.1.0 · experimental

Compress long context (retrieved RAG chunks or prior conversation turns)
by selectively pruning low-information tokens before the model sees
them. Implementations include LLMLingua, LongLLMLingua, and
Selective-Context.
