# Intent dispatcher

*processor* · `processor/intent-dispatcher` · v0.1.0 · experimental

Classify an incoming message into one of N intents and route to the
appropriate downstream pipeline. Backend can be a classifier rule
pack, a small local model, or a keyword router.

| industry | cross_industry |
