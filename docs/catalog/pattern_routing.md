# Routing

*pattern* · `pattern/routing` · v0.1.0 · stable

Classify the input and dispatch to the appropriate specialized
sub-pipeline. A classifier (LLM, embedder + threshold, or rule pack)
picks a route; the chosen sub-pipeline handles the task. Decouples
one front door from N specialized backends.
