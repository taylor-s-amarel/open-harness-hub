# Prompt-injection detector

*processor* · `processor/prompt-injection-detector` · v0.1.0 · experimental

Detect prompt-injection / jailbreak attempts in user input,
retrieved documents, or tool results. Two-tier: a fast regex /
classifier first pass plus an optional small-model classifier
second pass.
