# ISO country code normalize (alpha-2 / alpha-3 / numeric / name)

*processor* · `processor/iso-country-normalize` · v0.1.0 · stable

Resolve any country reference — alpha-2, alpha-3, numeric, English
name, French name, common alias — to canonical ISO 3166-1 codes.
Returns alpha-2, alpha-3, numeric, English short name, and the
ISO 3166-2 sub-division code if a region was specified.

Backed by the ISO 3166 data bundled at knowledge-pack/iso-country-codes.
