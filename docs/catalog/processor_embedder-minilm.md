# Text embedder (MiniLM-L6-v2)

*processor* · `processor/embedder-minilm` · v0.1.0 · stable

Generate 384-dimensional text embeddings using
`sentence-transformers/all-MiniLM-L6-v2`. Suitable for catalog
semantic search, RAG retrieval, and de-duplication. Replace with a
higher-dim embedder for production semantic retrieval.
