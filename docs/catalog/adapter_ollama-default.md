# Ollama (local default)

*adapter* · `adapter/ollama-default` · v0.1.0 · stable

Default local-Ollama transport. Configured for `llama3.1:8b` at
`http://localhost:11434`. Override by setting `OLLAMA_HOST` and
`OLLAMA_MODEL` env vars at runtime.
